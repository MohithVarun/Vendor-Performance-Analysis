# Vendor and Inventory Performance Analysis

## Description

This project analyzes sales and inventory data in the retail and wholesale sector to evaluate vendor performance, inventory efficiency, and overall profitability. The analysis helps identify operational gaps and supports better pricing, purchasing, and inventory management decisions.

## Business Objectives

- Identify underperforming brands that may require pricing or promotional adjustments
- Determine top vendors contributing to sales revenue and gross profit
- Analyze the impact of bulk purchasing on unit costs
- Assess inventory turnover to reduce holding and storage costs
- Compare profitability between high-performing and low-performing vendors

## Methodology

Exploratory data analysis is performed on sales, inventory, and vendor datasets using key business metrics such as gross margin, unit cost, and inventory turnover to evaluate performance and efficiency.

## Tools Used

- Python
- Pandas and NumPy
- Matplotlib and Seaborn
- Jupyter Notebook

## Key Insights

The analysis highlights vendor dependency risks, slow-moving inventory, and cost variations caused by purchasing patterns. These insights help businesses improve inventory planning, pricing strategies, and vendor management.

## Conclusion

This project demonstrates how data-driven analysis can improve inventory efficiency, reduce costs, and enhance profitability in retail and wholesale operations.
